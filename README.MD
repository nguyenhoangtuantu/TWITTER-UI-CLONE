# 😍 Twitter UI clone

Trang web này nhằm mục đích học tập và rèn luyện kỹ năng HTMl, CSS, và JavaScript bằng cách copy ý tưởng của web app X.com (được đổi tên thành Twitter) của Elon Musk.

![Ảnh giới thiệu về trang web](https://res.cloudinary.com/dhyljfnjx/image/upload/v1772435851/Screenshot_2026-03-02_141259_e7v8g5.png)

## 🤖 Tính năng

- Giao diện giống với X không có chút thay đổi nào.
- Bạn cũng có thể vào trang web này để học tập và rèn luyện kỹ năng flex và grid nhé.

## Cách cài đặt và sử dụng

1. Clone dự án: `https://github.com/nguyenhoangtuantu/TWITTER-UI-CLONE.git`
2. Mở file `index.html` trong trình duyệt của bạn để xem trang web.
